# SQL Customer Sales Analysis

This project demonstrates a customer sales analysis query using SQL.

## Skills Demonstrated

- Common Table Expressions CTEs
- INNER JOIN and LEFT JOIN
- GROUP BY aggregation
- CASE WHEN logic
- Window functions
- RANK
- NTILE
- LAG
- Customer segmentation
- Sales performance analysis

## Business Questions Answered

- Who are the top customers by total sales?
- Which product category does each customer buy the most?
- How many orders has each customer placed?
- Which customers are VIP, repeat, or occasional customers?
- How can customers be segmented by sales value?

## Notes

This is a sample portfolio SQL script using mock business data.
No private or real company data is included.
